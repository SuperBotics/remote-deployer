# Integration Examples

This folder provides practical integration examples for common developer and continuous integration workflows.

## Included Integrations
- `npm-script/` for direct terminal or package script deployment.
- `grunt/` for Grunt task integration.
- `webpack/` for post-build artifact deployment.
- `github-actions/` for deployment from a CI workflow.

Each example expects:
- `REMOTE_DEPLOYMENT_URL`
- `REMOTE_DEPLOYMENT_KEY`
- `REMOTE_DEPLOYMENT_PATH`
